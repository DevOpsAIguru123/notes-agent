# DevOps

Notes on containerization, CI/CD, and infrastructure automation.

## Topics

- **Docker Guide** - Container basics, Dockerfiles, and compose
- **CI/CD Pipeline** - Automated build, test, and deploy workflows

## DevOps Principles

1. **Automate everything** - If you do it twice, script it
2. **Infrastructure as Code** - Version control your infrastructure
3. **Shift left** - Catch issues early in the pipeline
4. **Monitor and observe** - You can't fix what you can't see
5. **Fail fast, recover faster** - Design for resilience
