# Database

Notes on relational and non-relational databases, query patterns, and data modeling.

## Topics

- **SQL Fundamentals** - Core SQL syntax, joins, and query optimization
- **NoSQL Concepts** - Document stores, key-value, and when to use what

## Choosing a Database

| Need | Best Fit | Examples |
|------|----------|----------|
| Structured data, transactions | Relational (SQL) | PostgreSQL, MySQL |
| Flexible schema, documents | Document store | MongoDB, CouchDB |
| Caching, sessions | Key-value store | Redis, Memcached |
| Relationships, graphs | Graph database | Neo4j, ArangoDB |
| Time-series metrics | Time-series DB | InfluxDB, TimescaleDB |
| Full-text search | Search engine | Elasticsearch, Meilisearch |
