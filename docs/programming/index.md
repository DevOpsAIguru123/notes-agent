# Programming

Notes on programming languages, patterns, and practical techniques.

## Topics

- **Python Basics** - Core syntax, data structures, and common patterns
- **JavaScript Tips** - Modern JS features, async patterns, and useful tricks

## General Principles

1. **Keep it simple** - Write clear code before clever code
2. **DRY** - Don't Repeat Yourself, but don't abstract too early
3. **Test first** - Write tests before or alongside implementation
4. **Read the docs** - Official documentation is your best friend
